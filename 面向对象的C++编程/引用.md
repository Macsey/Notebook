## 引用的基本用法：
**给变量起别名**
数据类型 &别名 = 原名
```cpp
int a = 10 ;
int &b = a;
```
## 引用的注意事项：
 1. 引用必须初始化：int &b;//这是错误的
 2. 引用一旦初始化之后就不可以更改 
 ```cpp
 int b=10;int c = 20;
 int &a=b;
 a = c;//注意这是赋值操作，并不是引用
 ```

## 引用和函数传递
1. 值传递：形参不影响实参
2. 地址传递：swap(&a,&b)  void swap(int * a ,int  * b)
3. 引用传递:swap(int & a,int & b) 注意：别名和原名可以相同
后两者形参都会修饰实参

## 引用作为函数的返回值
1. 不要返回局部变量的引用
2. 函数的调用可以作为左值

## 引用的本质
——是一个指针常量（指向是不可以修改的，指向内容的值是可以修改的）即初始化后就不可以发生改变
```cpp
int a = 0;
int &ref = a;//发现是引用系统自动变为int * const ref = &a;
ref = 20;//系统自动变为*ref = 20;
```

## 常量引用
作用：修饰形参，防止误操作

引用必须引用合法的内存空间
```cpp
//错误
int a & = 10;
//正确
const int a & =10;//变为只读，不可修改
//编译器将代码修改
int temp =10;
int & a = temp;
```

修饰形参：防止在函数中的误操作
```cpp
void(const int & a){
	错误：
	a = 10; 
}
```
