作用：函数名称相同，提高复用性
满足条件：
1. 同一个作用于下
2. 函数名相同
3. 函数参数类型不同，个数不同或者顺序不同
```cpp
#include<iostream>
using namespace std;
void print()
{ 
		cout<<"hello world"<<endl;
}
void print(int a)
{
	cout << "int a" << endl;
}
void print(double a)
{
	cout << "double a" << endl;
}
void print(int a,double b)
{
	cout << "int a,double b" << endl;
}
void print(double a, int b)
{
	cout << "double a, int b" << endl;
}
int main()
{
	print();
    print(1);
    print(1.0);
    print(1,2.0);
    print(2.0,1);
}
```
注意函数的返回值不能作为条件
