注意：
1. 如果某个位置已经有的默认参数，那么从这个值往后，从左到右都必须有默认值
2. 如果函数声明有默认参数，函数的实现就不能定义默认参数了（有歧义）即声明和实现只能有一个部分有默认参数
```cpp
int add(int a,int b=20,int c = 30)
{
	return a+b+c;
}
int main()
{
	add(10,30)//70  如果传入了自己的数据，就用自己的，如果没有，就用默认值
}
```
```cpp
#include<iostream>
using namespace std;
void print(int a,int b=10)
{ 
		cout<<a+b<<endl;
}
int main()
{
	int a = 10;
	int b = 30;
	print(a,b);
    
}
```
在有默认值的情况下传入一个值，这个值比默认值优先级高
